
PHIẾU THÔNG TIN TƯ VẤN - iOS style (Navy blue, LocalStorage, PDF, Ảnh khách hàng, 6 mục đúng form)

- Font size 22 (toàn app).
- 6 mục đúng như phiếu giấy:
  1. THÔNG TIN CÁ NHÂN
  2. TÌNH TRẠNG DA & SỨC KHỎE
  3. THÓI QUEN CHĂM SÓC & SINH HOẠT
  4. FACE MAPPING - ĐÁNH GIÁ VÙNG MẶT
  5. MỤC TIÊU CẢI THIỆN DA
  6. CAM KẾT & ĐỒNG Ý
- Ảnh khách hàng: nút "Chụp ảnh" (ẩn input file, preview lớn, canh giữa).
- Header: "PHIẾU THÔNG TIN TƯ VẤN"; nút "THÊM", "DANH SÁCH".
- Danh sách có nút "Sửa".
- Face mapping có đủ vùng: trán, giữa lông mày, mũi, má trong, má ngoài, cằm, đường viền hàm.

Cách chạy:
1. npm install
2. npm start

Deploy:
1. npm run build
2. Upload nội dung thư mục build lên host.
